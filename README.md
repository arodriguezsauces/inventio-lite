# Inventio Lite
Inventio Lite es un sistema de Inventario y Ventas de proposito general desarrollado con PHP y MySQL.

## Requisitos
PHP 7.4
MySQL 5.0 o superior

## Modulos
- Productos
- Categorias
- Caja
- Clientes
- Proveedores
- Inventario
- Usuarios
